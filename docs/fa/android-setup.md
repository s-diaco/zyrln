# راهنمای راه‌اندازی اپلیکیشن اندروید

اپلیکیشن اندروید، پروکسی رله را مستقیماً روی گوشی شما اجرا می‌کند. با این کار تمام ترافیک مرورگر گوشی شما از زنجیره رله عبور می‌کند و نیازی به روشن بودن کامپیوتر نیست.

## مسیر ترافیک

```
اپلیکیشن اندروید ← پروکسی محلی (در گوشی، پورت ۸۰۸۵)
                  ← اسکریپت گوگل (Apps Script)
                  ← رله خروجی (VPS)
                  ← سایت مقصد
```

این اپلیکیشن از سرویس `VpnService` اندروید استفاده می‌کند تا یک پروکسی HTTP سراسری در سیستم تنظیم کند که به پروکسی محلی (Go) متصل است.

## پیش‌نیازها

- اسکریپت گوگل راه‌اندازی شده باشد (به [README_FA.md](../../README_FA.md) مراجعه کنید).
- رله خروجی VPS در حال اجرا باشد (به [docs/fa/vps-setup.md](vps-setup.md) مراجعه کنید).
- نصب Go نسخه ۱.۲۵ یا بالاتر.
- نصب Android SDK به همراه NDK (نصب NDK از طریق Android Studio: بخش SDK Manager ← تب SDK Tools ← گزینه NDK).
- تنظیم متغیر محیطی `ANDROID_HOME` به آدرس SDK شما.

## بیلد و ساخت فایل نصب (Build)

ابتدا gomobile را نصب و آماده کنید:

```bash
go install golang.org/x/mobile/cmd/gomobile@latest
gomobile init
```

ساخت فایل APK:

```bash
make keystore   # ساخت کلید امضای اپلیکیشن (فقط یک بار اجرا شود)
make android    # بیلد کامل کدهای Go و اندروید و ساخت فایل نصب نهایی (Release)
# فایل خروجی در این مسیر خواهد بود: android/app/build/outputs/apk/release/zyrln-1.5.1.apk
```

یا ساخت نسخه دیباگ (بدون نیاز به کلید امضا):

```bash
make android-debug    # ساخت نسخه Debug APK
```

## نصب

فایل APK را به گوشی منتقل کرده و نصب کنید. ممکن است نیاز باشد اجازه نصب برنامه از منابع ناشناخته (Install unknown apps) را در تنظیمات اندروید بدهید.

## راه‌اندازی اولیه

### ۱. نصب گواهینامه امنیتی (CA Certificate)

اپلیکیشن ترافیک HTTPS را به صورت محلی رمزگشایی و رله می‌کند. مرورگر شما باید به گواهینامه محلی اعتماد کند.

۱. اپلیکیشن را باز کرده و دکمه **Install CA Certificate** را لمس کنید.
۲. فایل گواهینامه در پوشه `Downloads/zyrln-ca.pem` ذخیره می‌شود.
۳. در پنجره باز شده، **Open Settings** را بزنید.
۴. به بخش **Biometrics & security ← Other security settings ← Install from device storage** بروید. (در گوشی‌های مختلف ممکن است کمی متفاوت باشد).
۵. فایل `zyrln-ca.pem` را از پوشه دانلود انتخاب کنید.
۶. در صورت سوال، نوع آن را **CA certificate** انتخاب کنید.

این مرحله فقط یک بار انجام می‌شود.

⚠️ **نکته بسیار مهم:** هرگز فایل گواهینامه را از کامپیوتر به گوشی کپی نکنید. اپلیکیشن اندروید گواهینامه مخصوص به خود را می‌سازد و اگر از فایل ویندوز استفاده کنید، اتصال شما با خطای SSL مواجه خواهد شد.

### ۲. وارد کردن تنظیمات (Import Config)

در دسکتاپ، مطمئن شوید `config.env` تنظیم شده است، سپس دستور زیر را برای دریافت خروجی JSON اجرا کنید:

```bash
./zyrln -export-config
# خروجی چیزی شبیه به این خواهد بود: {"url":"https://script.google.com/...","key":"your-auth-key"}
```

این متن JSON را کپی کرده و در گوشی:
۱. گزینه **Import from Clipboard** را لمس کنید.
۲. کانفیگ به طور خودکار به لیست اضافه می‌شود.

**استفاده از چندین لینک گوگل:** برای پایداری بهتر وقتی یک لینک کند، مسدود یا بی‌سهمیه می‌شود، می‌توانید چندین لینک گوگل را در فیلد `url` با کاما (,) از هم جدا کنید:

```json
{"url":"link1,link2,link3","key":"your-auth-key"}
```

اپلیکیشن لینک‌های تنظیم‌شده را موازی امتحان می‌کند و اولین پاسخ موفق را استفاده می‌کند. بهتر است هر لینک مربوط به یک Apps Script جداگانه و یک اکانت گوگل جدا باشد.

### ۳. اتصال

۱. روی یکی از کانفیگ‌های لیست کلیک کنید.
۲. اجازه دسترسی VPN را در صورت درخواست بدهید.

دایره وضعیت سبز شده و رله فعال می‌شود. برای قطع اتصال، دوباره روی همان کانفیگ کلیک کنید.

### ۴. تست نهایی

مرورگر کروم را باز کرده و یک سایت فیلتر شده را چک کنید. باید بدون مشکل باز شود. اگر با خطای SSL مواجه شدید، مرحله ۱ (نصب گواهینامه) را دوباره چک کنید.

## محدودیت‌ها

- **فقط مرورگر:** این یک پروکسی HTTP است، نه یک VPN کامل. برخی اپلیکیشن‌ها (مانند اینستاگرام یا واتساپ) تنظیمات پروکسی سیستم را نادیده می‌گیرند و مستقیم متصل می‌شوند.
- نصب گواهینامه در برخی گوشی‌ها (مثل سامسونگ با اندروید ۱۰ به بالا) ممکن است کمی مراحل بیشتری داشته باشد.

برای مشاهده محدودیت‌های عمومی به فایل [README_FA.md](../../README_FA.md) مراجعه کنید.
